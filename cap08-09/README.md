# Projeto CSS Zen Garden

## Autor
Vitória Cestari Duarte

## Tema
Minimalismo Monocromático e Tipográfico

## Design de Referência
White Space / Clean Slate

## Link de Referência
https://www.csszengarden.com/218/

## Descrição
O projeto adota uma abordagem minimalista inspirada no design editorial suíço, onde o "espaço vazio" é utilizado como elemento ativo do layout. A estrutura macro utiliza CSS Grid para uma composição assimétrica, enquanto o Flexbox organiza os menus e alinhamentos internos. A paleta de cores monocromática garante foco total na tipografia.

## Recursos Utilizados
- CSS Grid Layout (Estrutura macro)
- Flexbox (Alinhamento de menus e rodapé)
- Variáveis CSS (:root) para consistência
- Media Queries (Responsividade)
- Propriedades tipográficas (line-height, letter-spacing)
