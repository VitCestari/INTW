3. Glossário Técnico

HTML
Linguagem de marcação para estruturar a web.
CSS
Linguagem de estilos para formatação visual.
Folhas de Estilo em Cascata.
JS
JavaScript
Linguagem de programação de alto nível usada no lado do cliente.
DOM
Interface de programação para documentos HTML e XML.
API
Interface de Programação de Aplicação.
Conjunto de rotinas e padrões para acesso a um software.
HTTP
HTTPS
Protocolos de transferência de hipertexto para comunicação web.
URL
Endereço de um recurso na rede.
DNS
Sistema que converte nomes de domínio em endereços IP.
